# CodeUp_basic100
파이썬 문법 복습

- `map`함수: map(function, iterable) <br>

- `sep` : separation 
```python 
print('s', 'e', 'p', sep = '@') #s@e@p
```

- `end`
```python
print("hello", end=" ")
print("world")
# hello world
```

- format
```python
print("{0}월{1}일 입니다.".format(8,2)) # 8월2일 입니다.
print("%s을 %d개 주세요."%("아이스크림", 10)) # 아이스크림을 10개 주세요.
```
